# 📊 E-commerce User Behavior & Revenue Analysis

## 🔍 Business Context

This project analyses user behavior and revenue patterns in an e-commerce environment. The goal is to identify key drivers of revenue and understand how user interactions differ across channels, devices, and regions.

## 🛠 Tools
	•	Python (Pandas, NumPy)
	•	Data Visualization (Matplotlib, Seaborn)
	•	SQL (BigQuery, CTEs, joins)
	•	Statistical Analysis (Mann-Whitney, Kruskal-Wallis, Chi-square)

## 📈 What I did
	•	Built a session-level dataset (349K rows, 19 features) by joining multiple tables
	•	Performed exploratory data analysis across channels, devices, countries, and product categories
	•	Investigated missing values and validated dataset structure
	•	Conducted statistical tests to evaluate differences in user behavior and conversion
	•	Analysed time series trends (daily & monthly revenue, seasonality)
	•	Explored correlations between key metrics

## 🎯 Business Value
	•	Supports data-driven marketing decisions
	•	Helps prioritize traffic growth strategies
	•	Prevents incorrect assumptions about channel effectiveness
	•	Highlights the importance of conversion-focused metrics
